Project Summary

To solve the "paradox of choice" on streaming platforms, we built a hybrid movie recommender system. It uses Collaborative Filtering (SVD) for deep personalization and Content-Based Filtering to effectively handle the cold-start problem for new users.

My Impact

My main impact was on the qualitative evaluation stage, where I acted as the "first user" to test the system's end-to-end flow. I critically analyzed the relevance of the recommendations, ensuring the transition from content-based to collaborative filtering felt logical and intelligent. My contribution ensured the model was not just technically accurate but also genuinely valuable and satisfying for the end-user.

What I Learned

This project taught me that a great machine learning model is only as good as the interface that delivers it. I learned how to translate complex mathematical concepts like SVD into a simple, intuitive user experience. This solidified my belief that true innovation lies at the intersection of powerful technology and human-centered design a philosophy I know is deeply valued.
